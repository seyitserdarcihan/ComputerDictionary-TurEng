Türkçesi 'dosya uzantısı'
İngilizcesi 'file extension'



İngilizce'de bu terimin üç şekilde kullanıldığı görülebilir:
filename extension
file name extension
file extension

Üçü de 'dosya uzantısı' demektir.


▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬ Genel Bilgiler ▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬  
Dosyaların uzantıları, dosya isminden nokta (.) ile ayrılır.
Örneğin bir metin dosyası (text file) şu uzantı ile gösterilir:
.txt
